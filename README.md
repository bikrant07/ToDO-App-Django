A clean and simple ToDo List web application built with Django that allows users to manage daily tasks efficiently. This project demonstrates CRUD operations, authentication handling, and Django best practices for beginners and intermediate developers.
| Technology           | Usage                     |
| -------------------- | ------------------------- |
| Python               | Core programming language |
| Django               | Backend framework         |
| SQLite               | Default database          |
| HTML/CSS             | Frontend                  |
| Bootstrap (optional) | Styling                   |
